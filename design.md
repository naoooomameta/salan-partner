# design.md — SALAN 認定パートナーサイト デザイン仕様

用途: 本サイト（LP＋パートナー向け詳細ページ）のデザイン言語を定義する仕様書。全ページこのトークン・規約に従うこと。

---

## 0. デザインの第一印象（方向性）

- **「信頼される日本のコンサル企業」トーン。** 明朝体の見出し＋狭いリーディングカラムで、堅実・格式・可読性を優先。
- **差し色はビビッドなオレンジ1色**（`#DB521F`）。CTA・強調・アイコンだけに使い、面では使わない。
- **背景は白とごく薄いグレーの2層**を交互に敷いてセクションを区切る（塗り分けで区切り、線や影は最小限）。
- 職人系・BtoB案件にそのまま転用しやすい「信頼設計」の骨格。

## 1. カラートークン

```css
:root {
  /* Brand / Accent */
  --c-primary:        #DB521F;  /* メインオレンジ（CTA・強調・アイコン） */
  --c-primary-dark:   #A63E17;  /* ホバー・グラデ下地 */
  --c-primary-bright: #F06734;  /* 明るいオレンジ（補助アクセント） */
  --c-accent-blue:    #1E53B5;  /* 副アクセント（リンク・図版） */
  --c-accent-red:     #BE1E1E;  /* 注意・強い訴求 */

  /* Premium（認定パートナー等の格上げ表現に限定使用） */
  --c-gold:           #F0CC96;  /* 濃紺/焦茶背景の上に置く金文字 */
  --c-gold-deep:      #9D7F51;
  --c-brown:          #846955;  /* プレミアム帯の背景 */

  /* Text */
  --c-text:           #202020;  /* 見出し・主要テキスト */
  --c-text-body:      #333333;  /* 本文 */
  --c-text-sub:       #666666;  /* 補足 */
  --c-text-mute:      #707070;  /* キャプション・注記 */

  /* Surface */
  --c-bg:             #FFFFFF;  /* 基本背景 */
  --c-bg-grey:        #F7F7F7;  /* 交互セクションの薄グレー（bg-grey） */
  --c-bg-grey2:       #F5F5F5;
  --c-border:         #E3E3E3;  /* 罫線・カード枠 */
}
```

**使用比率の目安**: 白 60% / 薄グレー 25% / テキスト系グレー 10% / オレンジ 5% 未満。オレンジは「押させたい所」だけ。

**CTAグラデーション（署名的表現）**:
```css
background-image: linear-gradient(-61deg, #DB521F 50%, #A63E17 0);
```
斜め61°でオレンジを2トーンに割るボタン/バッジ表現。多用せず主要CTAのみ。

## 2. タイポグラフィ

| 用途 | フォント | 補足 |
|---|---|---|
| 見出し・強調・数字 | **`"Noto Serif JP", serif`（明朝）** | 格式・信頼感の核。H1〜H3とキーメッセージに使う |
| 本文・UI | `"Noto Sans JP", sans-serif`（ゴシック） | 可読性重視の地の文 |
| 英字装飾 | `"Gentium Book Plus", serif` | 見出し脇の英語ラベル等の飾りに限定 |

- **base**: `font-size: 16px` / `line-height: 1.5`
- **ウェイト**: `400`(本文) / `500`(小見出し) / `700`(見出し・強調)。700が主役、400/500で階層。

### フォントサイズスケール（rem, base 16px)

| token | rem | px | 主な用途 |
|---|---|---|---|
| xs | 0.75 | 12 | 注記・キャプション |
| sm | 0.875 | 14 | 補足・ラベル |
| base | 1 | 16 | 本文 |
| md | 1.125 | 18 | リード文 |
| lg | 1.25 | 20 | 小見出し |
| xl | 1.375 | 22 | セクション見出し(sub) |
| 2xl | 1.5 | 24 | H2 |
| 3xl | 1.625 | 26 | H2(強) |
| 4xl | 1.875 | 30 | H1／メインコピー |

> 見出しは明朝(700)、本文はゴシック(400)。この2フォント使い分けがこのサイトの"らしさ"の8割。

## 3. レイアウト / スペーシング

- **リーディングカラム: `max-width: 766px`**（テキスト主体セクションの基準幅。狭く保って可読性を上げる）
- **中幅セクション: `999px` / `1100px`**（カード一覧・図版）
- **最大コンテナ: `1380px` / `1440px`**（フルワイド帯・ヒーロー）
- モバイルは左右 `16〜20px` のガター、単一カラム。
- セクションは `.section` 単位で縦リズムを刻み、**白 → 薄グレー(`#F7F7F7`) → 白** と交互配色で区切る（`bg-grey`クラス）。
- 区切りは「背景の塗り分け」で行い、境界線・ドロップシャドウは最小限（フラット寄り）。

## 4. コンポーネント指針

- **ボタン/CTA**: オレンジ塗り or 斜めグラデ。角はやや丸め、明朝でないゴシック太字。「無料相談」「資料ダウンロード」の2系統を各セクション末に反復。
- **見出しブロック**: 明朝の大見出し＋その下に細いオレンジのアクセント（下線/短いバー）＋ゴシックのリード文、の3点セット。
- **カード（メンバー/実績一覧）**: 白カード＋薄い枠(`#E3E3E3`)、写真上・名前(明朝太字)・肩書(オレンジ or グレー小文字)・説明(グレー本文)。3カラム→モバイル1カラム。
- **プレミアム帯**: 認定パートナーなど"格上げ"要素のみ、焦茶(`#846955`)背景＋金文字(`#F0CC96`)。乱用しない。

## 5. サイト構成

### index.html（募集LP）
1. **ヒーロー**（nav＋制度名＋分業リボン）
2. **problem**（`bg-grey`）— 課題3カード＋ブリッジ
3. **philosophy** — プレミアム帯（SALANの理念・頭字語）＋認定パートナー定義
4. **model**（`bg-grey`）— パートナー/本部の分業ダイアグラム
5. **ops** — 毎日課題の日次タイムライン
6. **benefits**（`bg-grey`）— ベネフィット4カード
7. **cta / sns / requirements / steps / faq**
8. **related**（`bg-grey`）— パートナー向け詳細ページ3カード
9. **entry**（プレミアム帯）— フォーム
10. **footer** — 関連ページリンク

### パートナー向け詳細ページ（To B）
- **program.html** — 制度詳細：職務範囲・認定基準・品質ガイドライン・認定プロセス
- **support.html** — 本部サポート体制：実行チーム・集客ファネル・日次運用・品質指標・ツール
- **economics.html** — 収益モデルの考え方：設計方針・収益構造・レバー・面談での開示項目

各詳細ページは共通の `site-head`（brand＋nav）／パンくず／`page-hero`／CTAストリップ／footer を持つ。
